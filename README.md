# VueWebApp
Vuejs version of consolidated Laravel web application

NOTE: still on development phase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
